Tokenfactory# Co Tien Lu Quan Reward Program
Project Overview
Co Tien Lu Quan is a chalet rental venue in Nha Trang, Vietnam.
This reward program introduces Pixalala (Pi) and CoTien Token (COTIEN) as internal loyalty points.

Token Information
Pixalala (Pi)
Contract Address: 0xC48488572eE79329ee19F25990C194cE400DCe01
Symbol: Pi
Decimals: 18
Total Supply: 1,000,000,000 Pi (initial), capped at 2,000,000,000 Pi
Usage: Earned from chalet rentals, redeemable for discounts.
CoTien Token (COTIEN)
Contract Address: 0xddfD69e796428896b5cA48d66E63A16761493e21
Symbol: COTIEN
Decimals: 18
Total Supply: 1,000,000,000 COTIEN (fixed, no minting)
Usage: Special loyalty points distributed via events, used for promotions.
